============================================
  每日待办事项清单  v1.0
  Daily Todo List
============================================

【使用方法】

  方式一（推荐）：双击 "启动清单.bat"
    → 独立窗口弹出，自动置顶（需要电脑安装了 Edge 浏览器）

  方式二：双击 "打开清单.bat"
    → 在默认浏览器中打开（100% 可用）

  方式三：直接双击 "TodoList.exe"
    → 启动置顶窗口 + 系统托盘图标

【功能介绍】

  ✅  勾选完成（带动画）
  ⭐  星标重要任务
  🎨  自定义文字颜色（20 色预设 + 取色器）
  📌  窗口强制置顶（始终在最前面）
  🏷️  自定义标签（可新建/删除）
  📅  日历视图（月度总览，点击跳转）
  📊  每日进度条
  🔍  搜索 + 筛选
  🔢  拖拽排序
  ⌨️  快捷键（Ctrl+F 搜索，Ctrl+Z 撤销删除）
  💾  导出/导入 JSON 备份
  🌙  夜间模式

【数据存储】

  所有数据保存在浏览器本地存储（localStorage）中，不上传网络。
  建议定期点击底部 "💾 导出" 备份数据。

【注意事项】

  - TodoList.exe 需要 .NET Framework 4.x（Win10 自带）
  - 置顶功能需要电脑安装了 Microsoft Edge 浏览器
  - 如果 exe 被 Windows 拦截，点击 "更多信息" → "仍要运行"
  - 首次运行 PowerShell 脚本可能需要确认执行策略（按 Y）

【分享给同事】

  把整个 "清单插件" 文件夹打包成 ZIP 发给同事即可，
  无需安装，解压即用。
